Moonwalker Odyssey: Android game developed with libGDX

Third project done for the second course of DAM.

Game developed in the Android Studio IDE and programmed in Java using the libGDX library.

You have 3 lives when you start the game.
If an enemy projectile hits you, you lose a life.
If an enemy collides with you, you lose a life.
For every enemy destroyed you get 50 points, but if an enemy manages to reach the bottom of the screen, you lose 50 points.
The game has no end and if you die you can play again.

Project developed by myself.
